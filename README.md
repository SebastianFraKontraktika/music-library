# Music Library

A simple desktop app built with [Avalonia UI](https://avaloniaui.net/) that lets you add and organize your music albums into custom categories.

## Features

- **add Albums** - quickly add albums with title, artist and cover art
- **Custom catagories** - Create and manage your own catagories to sort albums into, with a system to filter and a ranking system
- **Simple interface** - a simple and easy to understand interface

### Prerequisites

- [.NET 8 SDK](https://dotnet.microsoft.com/download) or later

### Fork and Run

1. Fork and clone the repository:
   ```bash
   gh repo fork https://github.com/SebastianFraKontraktika/music-library.git --clone
   cd music-library
   ```

2. Restore dependencies and run:
   ```bash
   dotnet restore
   dotnet run
   ```

## Usage

1. Launch the app
2. Click **New** to add a new album and fill in the details you wish, then click **Save**
3. Select the entry you wish to edit, then click **Edit** if you wish to edit *former* enteries
4. Click **Save** to save details, or **delete** if you wish to delete the selected entry
5. use the **filter** button above to filter through entries
